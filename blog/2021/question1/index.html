1. C#에서 struct와 class의 차이

July 27, 2021

  1. C#의 struct는 값(Value) 타입이며, 인스턴스는 Stack에 할당된다. 반대로 class는 참조(Refrence) 타입으로, 인스턴스는 Heap에 할당된다.
  2. struct는 인스턴스 선언 코드 블록이 끝나는 즉시 스택에서 해제가 된다. 클래스는 인스턴스 생성 선언 코드 블록 끝나도 바로 해제되지 않으며, 가비지 콜렉터(GC)에 의해서 해제가 이루어진다.
  3. struct는 상속이 불가능하며, class는 상속이 가능하다.
  4. struct 선언할 때, 접근지시자를 정해주지 않으면 public으로 선언되고 class는 private로 선언된다.
  5. struct는 기본 생성자(Default Constructor)를 정의할 수 없으며, 소멸자도 정의할 수 없다.
  6. struct는 new 연산자 사용않고, 선언하는 것만으로도 인스턴스가 생성된다.

추가

  1. struct과 class 둘 다 메소드, 프로퍼티, 필드를 가질 수 있다.
  2. struct가 가진 멤버로 class를 가질 때, 그 class의 인스턴스가 생성되면 힙에 객체가 할당이 되고, 스택에 그 객체의 메모리 주소값이 할당이 된다.
  3. Value타입의 배열을 생성하면, 스택에는 배열의 주소값이 할당되고, 힙에 배열의 요소들이 할당된다.
